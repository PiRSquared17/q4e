Build Plugin and Plugin Management

The Plugin and Plugin Management allows you to modify the plugins for the project. Plugins contain information being configured for a particular project build. Plugin Management works the same way as the plugins, however, it is intended to configure project builds that inherit from it.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add/edit plugin and plugin management:

  1. Right click on the element in the tree and select a task: add, and edit
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a continuous management and mailing list:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To delete the element, right click on it and select delete task from the menu list.
