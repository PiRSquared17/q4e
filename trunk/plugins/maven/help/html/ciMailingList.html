CI Management and Mailing Lists

Continuous Integration is a scheduled build based on triggers or timings (such as hourly or daily). Mailing lists are great tool for keeping in touch with people working or using the project.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a continuous management and mailing list:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a continuous management and mailing list:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a continuous management and mailing list, select the continuous management and mailing list from the list and click Remove.
