Update Snapshot JARs

The Update Snapshot JARs feature updates the versions of the artifact dependencies of the project. This process involves downloading the latest version of the artifact from the configured remote repository and putting it in your local repository.

To update the snapshot JARs:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven2 menu options.
  2. Select Update Snapshot JARs.
