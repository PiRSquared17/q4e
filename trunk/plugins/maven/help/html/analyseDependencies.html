Analyse Dependencies

The Analyse Dependencies view shows a tree view of a project's direct and transitive dependencies which can be used in debugging common dependency problems.

This feature is found in the Maven 2 Menu which allows you to do the following:

  • Exclude all instances of a selected dependency.
  • Force the version of a given artifact.

Executing the options will alter the <dependencyManagement> and the <dependency> sections of the project's pom.xml.
