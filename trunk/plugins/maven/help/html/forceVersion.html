Force Version

The Force Version option utilizes Maven's <dependencyManagement> section of the project's pom.xml to ensure that the version of the selected artifact is a specific version or a version within a certain range.

To select a specific version for an artifact:

  1. Right click on the selected artifact and select Force Version from the menu.
  2. A popup will be displayed with the list of all the other versions present in the dependency tree. Select the version to use or type the version range in the Other text field.
    The item marked as Currently Selected is the version that Maven has currently resolved.
  3. Click OK.
    Selecting Cancel will cancel the operation without making any changes to the project's pom.xml.
