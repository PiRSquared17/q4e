Deploying Artifacts

The Deploy Artifacts option on the Maven2 Menu runs the standard Maven mvn deploy command. The deploy lifecycle goal deploys the distributable archive into the remote Maven repository configured in the distributionManagement section of the POM.

To deploy artifacts:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven2 menu options.
  2. Select Deploy Artifact.
