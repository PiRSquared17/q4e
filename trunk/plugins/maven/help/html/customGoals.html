Using Custom Goals

Creating a Custom Goal

  1. Right-click on the name of your project, then select Run As > Run Configurations.... The Create, manage, and run configurations window appears.
  2. Right-click on Maven 2 on the left side of the screen, then select New to display a new input page on the right side of the screen
  3. Type a name for the new custom goal/configuration into the Name field at the top of the input screen.
  4. Either type the name of your project into the Project field, or click on the Browse button to locate and select your project.
  5. Type the type of Maven goal into the Goal field (for example, install, test, or package).
  6. Click on the New button to enter a property for the new custom goal. A new dialog opens with fields for the key and the value
  7. Type the Key and Value for the property, then click on OK to add them to the list for the new goal.
    While you are adding properties, you can use the Edit button on the right of the screen to make changes to a property you added, or use the Remove button to delete a property from the list. To remove all of the goals you added, click on the Revert button.
  8. Click on Run to run the new goal you created, then Close to exit.
    To run the goal later, right-click on the project file in the Package Explorer window, then select Run As > Maven 2. From the dialog box, select your custom goal and click OK.

Executing Custom Goal

To run a custom goal:

  1. In the Package Explorer window, right-click on the project file for which you created the custom goal.
  2. Select Run As > Maven 2.
  3. In the dialog box, select the custom goal you want to run and click OK.
