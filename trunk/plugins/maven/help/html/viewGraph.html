Viewing a Dependency Graph

The View Dependencies Graph option from the Maestro Eclipse Integration (Maven 2) menu displays a detailed, graphical representation of project dependencies. The graph includes several options that allows you to adjust how the graph appears, as explained below.

To display the dependencies graph:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven 2 Menu options.
  2. Click the View Dependencies option. After a few seconds, a window appears, showing the dependencies graph.

Dependency Graph Representation

Box Color Representation
Green Represents the project. It is displayed in the center.
Blue Runtime dependencies.
Dark Yellow Test-scope dependencies.
Yellow Selected dependency.

The arrows between boxes show the dependencies including the version. Thicker lines represent direct dependencies, while the thinner ones represent transitive dependencies (dependencies of dependencies) for the project.

It is possible to reorder the boxes by using drag and drop.
