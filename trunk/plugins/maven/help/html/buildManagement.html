Build Management

The Build Management contains the base set of elements of <build> configurations from the project pom.xml.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a extension and filter:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a extension and filter:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a extension and filter, select the extension and filter from the list and click Remove.

To modify the build elements, supply the values for the fields to be changed and save the file for the changes to take effect.
