Processing Modules Recursively

The Process Modules Recursively feature allows you to enable/disable the recursive processing of the project's modules. By default, this feature is disabled.

To enable/disable feature:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven2 menu options.
  2. Select Process Modules Recursively.
    A check mark beside the option indicates that the feature is enabled, otherwise, the feature is disabled.
    This feature works the same way as the Execute recursively option in Global Preferences.
