XML Automatic Complete Suggestion

The XML automatic complete suggestion feature provides suggestions to possible configurations to be added or modified in the Maven 2 pom.xml.

To use this feature, open the pom.xml in a XML editor and right-click on the row you wish to add or modify configuration and select the option from the menu. The following table lists the suggestions for automatic completion.

Automatic complete Options

Options Description
Remove Removes the selected row.
Add DTD Information... Add another DTD to be used.
Edit Namespaces... Allows add, edit, and delete of namespaces.
Add Child Adds sub-properties. For example, outputDirectory and sourceDirectory can be added under builds property.
Add Before or Add After Adds properties before or after the selected row. The following enumerates some of the properties:
  • ciManagement
  • dependencyManagement
  • inceptionYear
  • modules
  • parent
  • pluginRepositories
  • prerequisites
  • profiles
  • properties
  • reports
  • comments
  • Add processing instruction
