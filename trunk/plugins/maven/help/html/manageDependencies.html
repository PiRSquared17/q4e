Managing Dependencies

The Manage Dependencies option on the Maven 2 Menu from the allows you to view and edit dependencies inside a project.

To use the Manage Dependencies option:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven 2 Menu options. A window opens, showing a list of all project dependencies. From this window, you can choose to add, edit, or remove dependencies.
  2. The following table explains the steps for adding, editing, or removing dependencies:

Manage Dependencies

To do this: Follow these steps:
Add project dependencies,
  1. Click New.
  2. In the window that opens, fill in the Group ID, Artifact ID, Version, and Scope. Or click ... to search.
  3. Click OK to close the window and create the new dependency.
Edit a project dependency,
  1. Click to select the dependency that you would like to edit.
  2. Click Edit. Or click ... to search.
  3. In the window that opens, make any changes you want to the dependency.
  4. Click OK to accept the changes and close the window.
Search for a dependency,
  1. While in Add or Edit, click ...
  2. In the window that opens, fill in the text for the Group ID, Artifact ID, or Version you would like to locate in your local repository.
  3. Click Search. Your local Maven repository will be searched, which is typically located here: [user_home]/.m2/repository.
  4. Click on the artifact you would like to use.
  5. Click OK to use the selected artifact and close the window.
Remove a project dependency,
  1. Click to select the dependency you would like to remove.
  2. Click Remove.
  3. Click OK to accept the removal and close the window.
