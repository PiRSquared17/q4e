Properties and Modules Management

The Properties are the constant values being configured and used in the project. The Modules are the sub-projects included in the project.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a property and module:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a property and module:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a property and module, select the property and module from the list and click Remove.
