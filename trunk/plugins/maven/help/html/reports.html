Generating Reports

The Generate Reports option runs the traditional Maven mvn site command and lets you generate various reports and lists with information from the POM. By default the following reports and lists are generated:

  • Dependencies Report
  • Mailing Lists
  • Continuous Integration
  • Source Repository
  • Issue Tracking
  • Project Team
  • License

To generate the reports:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven 2 Menu options.
  2. Select Generate Reports. The reports generate and are placed in a site directory under the target directory. For example: [equinox_home]/target/site.
