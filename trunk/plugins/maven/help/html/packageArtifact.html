Packaging an Artifact

The Package Artifacts option runs the traditional Maven mvn package command. The package lifecycle goal assembles the tested application code and resources into a distributable archive.

To package an artifact into the local Maven Repository,

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven 2 Menu options.
  2. Select Package Artifact.
  3. View the output pane to see the location of your new archive.
