Searching Artifacts

This section discusses the configuration of Artifact Search feature. The Artifact Search feature allows you to register and use search engines for indexing solutions. This feature is in Eclipse Preferences window found in Window > Preferences.

To register search engine to use:

  1. In the left part of the window, double-click to expand Maven. Select Artifact Search in the list below Maven.
  2. Select the box beside the Search Provider Name from the right side of the screen.
  3. Click Apply. This will apply the configurations immediately.
  4. Click OK.

To add a search engine to the list of Search Providers:

  1. In the left part of the window, double-click to expand Maven.
  2. From the left side of the window, double click to expand Artifact Search.
  3. Select Lucene Artifact Search Preferences.
  4. From the display on the right part of the window, click Add.
  5. Fill in the necessary fields in the pop-up window.
  6. Click OK.
