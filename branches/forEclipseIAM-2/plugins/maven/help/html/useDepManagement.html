Use Maven Dependency Management

The Maestro Eclipse Integration Plugin lets you manage dependencies through Eclipse or through Maven. The table, below, explains how you choose between Eclipse and Maven.

Dependency Management through Maven

If: Then:
you select Enable Maven Dependency Management, Maven uses information inside the pom.xml file to conduct dependency management. Changes to the pom.xml file will trigger a change in the Eclipse project classpath.
you select Disable Maven Dependency Management, Eclipse handles dependency management as usual.

To enable or disable Maven dependency management:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven 2 Menu options.
  2. Select Use Maven Dependency Management.
