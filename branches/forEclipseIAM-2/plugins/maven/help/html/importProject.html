Importing Maven 2 Projects

The Eclipse IDE includes an Import Wizard that lets you import projects of different types, but did not provide for Maven 2 projects. To import a Maven 2 project, you had to issue a command from inside the project you wanted to import (the mvn eclipse:eclipse command) prior to importing it. This command would take the Maven project and generate the necessary Eclipse project files. However, the new version of the Eclipse Integration plugin lets you use the Eclipse Import Wizard to import Maven 2 projects without having to execute the mvn eclipse:eclipse goal.

To import a Maven 2 project or projects:

  1. From the Eclipse main menu, select File > Import to display the Import Wizard. The Import Wizard shows a window in which you can select the import source.
  2. Click to expand the General input source (if it is not already expanded). Then, double-click on Maven 2 Project to display the Maven 2 Project Import window.
  3. Click on the Browse button, then locate the directory that contains the Maven 2 project or projects you want to import. After you select the project directory, the available Maven 2 projects appear in the Import Wizard window.
  4. Click to select the box in front of the project or projects you want to import. On the right side of the Import Wizard window are buttons that provide additional functionality when you are selecting your project or projects for import. The table, below, explains the different ways you can select, deselect, or refresh the list of projects.

    Options in the Maven 2 Import Wizard Window

    If: Then:
    you want to select more than one project for import, click the box in front of each project.
    you want to select all projects that display in the Import Wizard window, click on the Select All button on the right side of the Wizard window.
    deselect a single project, click to remove the check mark from its selection box.
    deselect all projects you selected, click the Deselect button on the right side of the Wizard window.
    refresh to display additional projects in the project directory, click on the Refresh button on the right side of the Wizard window.
  5. After you have selected the project or projects you want to import, click on the Finish button. The system imports the Maven 2 project or projects, which then appear in the Package Explorer window. Corresponding confirmation messages appear in the message window at the bottom of the screen.
