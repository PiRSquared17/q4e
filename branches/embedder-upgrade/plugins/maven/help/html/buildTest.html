Build Test Resources

The Build Test Resources are elements just like the resources only that these elements are used during the test phases.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a test resource, includes, and excludes:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a test resource, includes, and excludes:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a test resource, includes, and excludes, select the test resource, includes, and excludes from the list and click Remove.
