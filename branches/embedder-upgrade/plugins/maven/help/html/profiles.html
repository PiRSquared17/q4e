Activating and Deactivating Profiles

The Profile View displays the list of profiles configured. The selected profile from the list indicates the profile used or the active profile. Alternately, you can also activate/deactivate the profile used from the list by ticking the box before the name.

To view the the list of profiles:

  1. From the pop-up window in Window > Show View > Other, expand Apache Maven menu by double clicking.
  2. Select Profile from the options in Apache Maven.
  3. Click OK.
