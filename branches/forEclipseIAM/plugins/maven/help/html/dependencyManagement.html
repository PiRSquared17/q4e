Dependency Management

The Dependency Management section allows you to manage dependencies and configure its exclusions.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add new dependency or exclusion:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a dependency or exclusion:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a dependency or exclusion, select the dependency or exclusion and click Remove.
