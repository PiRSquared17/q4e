Executing Goals

The Execute Goal option from the Maven 2 Menu allows you to run Maven 2 commands from the Eclipse Integrated Development Environment.

To execute Maven 2 command:

  1. Right-click on the name of your project, then select Maven 2 Menu > Execute Goal.
  2. Fill in the necessary fields from the popup window:
    • Goal: This is the Maven 2 command to run.
    • Goal Properties: Other properties to use when running the goal. You can add, edit, and remove the properties.
  3. Click OK to execute the goal.

Executing Custom Goal

To run a custom goal:

  1. In the Package Explorer window, right-click on the project file for which you created the custom goal.
  2. Select Run As > Maven 2.
  3. In the dialog box, select the custom goal you want to run and click OK.
