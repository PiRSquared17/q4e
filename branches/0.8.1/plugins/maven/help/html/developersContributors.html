Developers and Contributors

The Developers/Contributors sections in the POM file list the persons who contributed to the project. The developers are those who have commit rights to the project while the contributors does not have any commit rights.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a developer, contributor, and properties:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a developer, contributor, and properties:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a developer, contributor, and properties, select the developer, contributor, and properties from the list and click Remove.
