Fetching Source JARs

The Fetch Source JARs feature downloads the source JAR of the project's dependencies and place it in the local repository. The source JAR naming format is [artifact_name]-[version]-sources.jar. For example if the artifact's named is commons-logging-1.0.4.jar, the source JAR would be commons-logging-1.0.4-sources.jar.

To download the source JAR of the artifact:

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven2 menu options.
  2. Select Fetch Source JARs.
