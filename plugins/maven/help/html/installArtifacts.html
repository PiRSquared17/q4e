Locally Installing an Artifact

The Locally Install option runs the traditional Maven mvn install command. The install lifecycle goal installs a distributable archive into the local Maven Repository.

To deploy an artifact into the local Maven Repository,

  1. In the Package Explorer window, right-click on the folder that contains your project, then select Maven 2 to display the Maven 2 Menu options.
  2. Select Locally Install Artifact.
  3. The artifact(s) will be deployed to your local Maven repository, which is typically here: [user_home]/.m2/repository.
