POM Editor

The POM Editor feature is a form-based editor for Maven 2 projects. This feature allows managing the contents of the pom.xml by providing the values for the certain project POM elements.

The following are the Maven 2 POM elements supported:

  • Project Information
  • Dependencies/Dependency Management
  • Licenses
  • SCM
  • Organization
  • Issue Management
  • Developers/Contributors
  • Properties/Modules
  • Build Management
  • Build Resources
  • Build Test Resources
  • Build Plugin/Plugin Management
  • CI Management
  • Mailing Lists
  • Repositories
  • Distribution Management

For a complete reference on Maven 2 POM elements, refer to http://maven.apache.org/pom.html.

To use the POM editor, right click on the pom.xml file of the project and select Open With > Maven 2 Pom Editor. Maven 2 Pom Editor will then be displayed.


Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.
