Repositories

Repositories are collections of artifacts which adhere to the Maven 2 repository directory layout with the POM structure of [base_repository]/groupId/artifactId/version/artifactId-version.pom. This is where artifacts are collected and stored.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a repository:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a repository:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a repository, select the repository from the list and click Remove.
