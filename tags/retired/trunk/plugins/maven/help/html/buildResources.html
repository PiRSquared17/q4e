Build Resources

The build resources is an element inside the <build> configuration. This contains the resources configuration, and the location of the resources within the project.

Modifying the values in the pom.xml does not automatically save the changes. You need to save the POM file to make the changes take effect.

To add a resource, includes, and excludes:

  1. Click New.
  2. Fill in the values for the fields in the pop-up window.
  3. Click OK.

To edit a resource, includes, and excludes:

  1. Click Edit.
  2. Modify the values for the fields in the pop-up window.
  3. Click OK.

To remove a resource, includes, and excludes, select the resource, includes, and excludes from the list and click Remove.
