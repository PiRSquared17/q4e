Exclude All

The Exclude All option utilizes Maven's <dependencyManagement> and the <dependency> sections of the project's pom.xml to ensure that the selected artifact is completely removed from the dependency tree.

To remove an artifact from the dependency tree:

  • Right click on the selected artifact and select Exclude All option from the menu.
  • Click OK.
    Selecting Cancel will cancel the operation without making any changes to the project's pom.xml.
